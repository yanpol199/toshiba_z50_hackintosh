# toshiba_z50_hackintosh
loader for start mac os

## Hardware

| Type          | Name                                            |
| ------------- | ----------------------------------------------- |
| CPU           | Intel i5\i7 4xxxU                               |
| RAM           | 2xDDR3l 1600MHz                                 |
| GPU           | Intel HD Graphics 4400                          |
| Drive         | mSATA SSD                                       |
| Sound card    | Realtek                                         |
| Wireless card | Apple Broadcom BCM94360CS2                      |
| Display       | LVDS display 1366x768(for fullHD del AAPL00)    |
